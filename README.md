# Projet_IGSD_Formule3

Petit prototype de jeu de course en vue a la premiere personne sous Processing.

## Commandes

- `W` / `S` ou fleches `haut` / `bas` : accelerer et freiner
- `A` / `D` ou fleches `gauche` / `droite` : changer de file
- `N` : activer / desactiver le mode nuit
- `F` : activer / desactiver le brouillard
- `R` : reinitialiser la course

## Contenu actuel

- skybox et sol texturés
- circuit genere a partir de points de controle et de courbes de Bezier
- texture de route generee pixel par pixel
- voiture du joueur et voiture adverse
- camera a la premiere personne
- mini-carte affichee en haut a gauche
- mode nuit avec phares
- brouillard simple base sur la distance
